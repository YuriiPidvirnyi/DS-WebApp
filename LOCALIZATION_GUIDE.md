## 🎯 Загальні принципи

### 1. **Цільова аудиторія**

- Українськомовні пацієнти різного віку
- Переважно міські жителі Києва та регіону
- Різний рівень технічної грамотності

### 2. **Основні принципи перекладу**

- ✅ **Природність**: Використовуємо живу українську мову, а не кальки з російської
- ✅ **Зрозумілість**: Уникаємо занадто складних медичних термінів без пояснення
- ✅ **Консистентність**: Один термін = один переклад по всьому сайту
- ✅ **Довіра**: Професійна, але доступна мова

### 3. **Загальні правила**

- Використовуємо українську "и" замість російської "ы"
- Переважаємо українські відповідники іноземним словам: "зуб" замість "зуба"
- Використовуємо сучасну українську орфографію (2019 року)

## 💬 Тон і стиль мовлення

### **Характеристики тону**

- **Професійний**, але не холодний
- **Дружній**, але не панібратський
- **Впевнений**, але не зарозумілий
- **Турботливий** та **емпатичний**

### **Звертання до пацієнтів**

- ✅ Використовуємо "ви" (ввічливе звертання)
- ✅ "наші пацієнти", "ваше здоров'я", "ваша посмішка"
- ❌ Уникаємо "клієнти" - кажемо "пацієнти"

### **Приклади правильного тону:**

**✅ Добре:**

- "Ми дбаємо про ваше здоров'я та комфорт"
- "Наші досвідчені лікарі проведуть детальну діагностику"
- "Зв'яжіться з нами зручним для вас способом"

**❌ Погано:**

- "Наші клієнти отримують послуги"
- "Ми надаємо медичні маніпуляції"

## ⚙️ Технічні стандарти

### **Структура локалізаційних файлів**

```typescript
// src/locales/uk.json - основний файл локалізації
{
  "navigation": { ... },    // Навігація сайту
  "common": { ... },        // Загальні елементи
  "buttons": { ... },       // Тексти кнопок
  "forms": { ... },         // Форми та валідація
  "services": { ... },      // Послуги та процедури
  "meta": { ... }          // SEO метадані
}
```

### **Використання хука useLocale**

```typescript
import { useLocale } from '@/hooks/useLocale'

// В компоненті
const { t, tArray, formatDate } = useLocale()

// Отримання перекладу
const title = t('home.hero.title') // "Ваша посмішка - наша місія"

// З параметрами
const rating = t('testimonials.rating', { rating: '4.9' })
// "Рейтинг: 4.9 з 5 зірок"

// Масив перекладів
const benefits = tArray('home.hero.benefits')
// ["Безболісне лікування", "Гарантія на всі послуги", ...]
```

### **HTML та мета-теги**

```html
<!-- index.html -->
<html lang="uk">
  <title>Dental Story - Сучасна стоматологічна клініка в Києві</title>
  <meta name="description" content="Професійна стоматологічна допомога..." />
  <meta property="og:locale" content="uk_UA" />
</html>
```

## 🔄 Процес оновлення локалізації

### **Додавання нового тексту**

1. Додайте ключ до `src/locales/uk.json`
2. Використайте хук `useLocale()` у компоненті
3. Перевірте, що TypeScript не видає помилок
4. Протестуйте на різних пристроях

### **Зміна існуючого перекладу**

1. Знайдіть ключ у `src/locales/uk.json`
2. Оновіть переклад
3. Перевірте всі місця використання
4. Протестуйте UI/UX
